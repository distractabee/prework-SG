HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The a tag stands for anchor and is used to add hyperlinks
  • The img tag is used to add images and can contain a link to the image, alt text and dimensions
  • Header tags go from h1 to h6 in order of size. h1 can only be used once on a page.

CSS

  • CSS is used to add style to a webpage
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • It is also possible to embed an entire stylesheet directly in the HTML file using a style element in the head element.
  • This internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • When using an external style sheet, we link the file using a link element placed in the HTML's head.
  • It allows developers to keep all their CSS rules in a separate file, which makes design changes easier.

Git

  • Github is a cloud-based coding service
  • With Github you can easily work collaboratively
  • Git Bash allows you to work on your local Windows OS computer
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url] retrieve an entire repository from a hosted location via URL

JavaScript

  • JavaScript is used to add functionality/interactivity to a webpage
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Defining a function will not execute it in the code automatically.
  • Functions must be called in the code to execute.
  • A function is a reusable block of code that completes a specific task.
